# Ejercicio 17 – Operaciones Booleanas en Blender
Objeto asignado

Este repositorio contiene los modelos exportados en formato .obj correspondientes a
las tres operaciones booleanas aplicadas en Blender:

- Unión.obj
- Diferencia.obj
- Intersección.obj

## Versión utilizada
- Blender 4.x
- Formato: Wavefront OBJ
